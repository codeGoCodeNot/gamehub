# GameHub

A React game discovery app built with modern web technologies.

**Features:**

- Browse 500k+ games from RAWG API
- Filter by genre & platform
- Search, sort by release date/rating
- Game details: trailers, screenshots, metacritic score
- Dark mode support
- Infinite scroll pagination

**Tech Stack:** React 19, TypeScript, Vite, React Query, Zustand, Tailwind CSS, React Router
